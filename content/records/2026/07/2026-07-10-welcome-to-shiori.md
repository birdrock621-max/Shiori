Shiori begins with a simple promise: the writing stays yours. Each entry is an ordinary Markdown file stored in your GitHub repository.

## What this archive is for

Use it for journals, finished work, lessons, project decisions, walks, reading notes, or anything else you want to return to later.

- [x] Created the first shelf

## A portable foundation

There is no database and no hosted editor to leave behind. The source remains readable even without the website.
