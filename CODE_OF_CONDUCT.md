# 行動規範

Shioriのコミュニティでは、経験、年齢、身体、国籍、性別、性自認、性的指向、宗教、経済状況、技術水準にかかわらず、すべての参加者が安心して関われる場を目指します。

## 期待する行動

- 相手の経験と時間を尊重する。
- 人ではなく、提案・コード・結果について建設的に話す。
- 初心者にも分かる言葉で背景と理由を説明する。
- 間違いを認め、修正し、他者の改善を助ける。
- 個人の日記や公開範囲に関する慎重さを尊重する。

## 許容しない行動

- 侮辱、差別、脅迫、執拗な接触。
- 性的な言動や、望まれていない個人的関心。
- 個人情報、非公開の連絡、秘密情報の公開。
- 他者を危険にさらす行為や、コミュニティ運営を妨げる行為。

適用範囲は、Issue、Pull Request、Discussion、その他Shioriを代表して参加する場です。問題の報告は公開の場で当事者の情報を広げず、リポジトリ管理者へ非公開で連絡してください。管理者は状況に応じて、注意、投稿の削除、一時的または恒久的な参加停止を行います。

この文書は [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) の考え方を基に、Shiori向けに簡潔にしたものです。
